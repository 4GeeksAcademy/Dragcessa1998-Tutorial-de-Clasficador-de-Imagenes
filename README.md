# Sistema de Clasificación de Imágenes: Perros vs Gatos

Proyecto de clasificación de imágenes desarrollado como práctica de Machine Learning y Deep Learning. El objetivo fue construir un flujo completo para identificar imágenes de perros y gatos utilizando una red neuronal convolucional tipo VGG con TensorFlow/Keras.

## Proceso realizado

- Carga y validación del dataset.
- Visualización inicial de imágenes de perros y gatos.
- Redimensionamiento de imágenes a 200x200 píxeles.
- Preparación de datos de entrenamiento y prueba.
- Creación de generadores de imágenes con Keras.
- Construcción de una CNN tipo VGG.
- Uso de callbacks como ModelCheckpoint y EarlyStopping.
- Evaluación del mejor modelo guardado.
- Visualización de métricas mediante gráficos.
- Predicción final sobre una imagen de prueba.

## Tecnologías utilizadas

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Pillow
- Jupyter Notebook

## Resultado

El notebook principal queda ejecutado y documentado para facilitar la revisión del proceso completo, desde la carga de imágenes hasta la evaluación del modelo.
